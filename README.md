# Ideate Industrial Design

Plateforme de conception industrielle assistée par IA avec Next.js 16.0.0 et Supabase.

## Fonctionnalités

- 🔐 Authentification sécurisée avec Supabase
- 🎨 Génération de concepts de produits avec IA
- 📊 Tableau de bord interactif
- 🔄 Processus de design itératif
- 📱 Interface responsive

## Technologies

- **Frontend**: Next.js 16.0.0, React 18, Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Auth, Storage)
- **Authentication**: Supabase Auth
- **Styling**: Tailwind CSS

## Installation

1. Cloner le projet
2. Installer les dépendances:
   \`\`\`bash
   npm install
   \`\`\`

3. Configurer les variables d'environnement:
   Copier le fichier \`.env.local.example\` vers \`.env.local\` et remplir les valeurs

4. Lancer en mode développement:
   \`\`\`bash
   npm run dev
   \`\`\`

5. Ouvrir [http://localhost:3000](http://localhost:3000)



## Scripts disponibles

- \`npm run dev\` - Lance le serveur de développement
- \`npm run build\` - Build la version production
- \`npm run start\` - Lance le serveur production
- \`npm run lint\` - Exécute ESLint

## Déploiement

Le projet peut être déployé sur Vercel, Netlify, ou toute autre plateforme supportant Next.js.
